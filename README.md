# 📝 To-Do List Web App

A simple and interactive **To-Do List Web Application** that allows users to add and manage daily tasks efficiently. This project is built using basic web technologies and is perfect for beginners learning JavaScript and DOM manipulation.

---

## 🚀 Features

* ➕ Add new tasks
* 📋 Display tasks dynamically
* ❌ Remove tasks (if implemented in JS)
* 🎯 Simple and clean UI
* 📱 Responsive design

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript** – Functionality and interactivity

---

## ⚙️ How to Run

1. Download or clone the repository
2. Make sure all files (`index.html`, `style.css`, `script.js`) are in the same folder
3. Open `index.html` in your browser

---

## 💡 How It Works

* User enters a task in the input field
* Clicks the **"Add Task"** button
* Task gets added to the list dynamically using JavaScript
